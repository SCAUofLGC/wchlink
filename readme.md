# WeAct-wchlink

## 简介
WCH-Link模块可用于沁恒RISC-V架构MCU在线调试和下载，也可用于带有SWD接口的ARM 内核MCU的在线调试和下载。同时带有1路串口，方便调试输出。 mini版本v1.0
